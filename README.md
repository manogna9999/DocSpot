# DocSpot-Seamless Appointment Booking for Health

# Project Description
This is a full-stack web application developed using the MERN stack. 
The system allows users to register, browse doctors, and book appointments.
Doctors can manage appointments and update status.
Admins can approve doctors.

# Tech Stack
- MongoDB
- Express.js
- React.js (TypeScript + MUI)
- Node.js
- JWT Authentication
- Redux Toolkit

# Features
- User Registration & Login
- Doctor Registration & Profile
- Admin Approval System
- Appointment Booking
- Appointment Status Tracking
- Secure JWT Authentication
- Role-based Access Control
